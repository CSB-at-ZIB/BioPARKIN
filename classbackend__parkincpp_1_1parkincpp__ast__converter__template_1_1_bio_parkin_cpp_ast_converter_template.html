Detailed Description

Implements the BaseAstConverterTemplate and fills all the predefined methods with actual code to convert libsbml ASTNodes into PARKINcpp Expression objects.

Parameters:
astConverter,:The reference to the ASTConverter is needed so that recursive calls along the node tree are possible. astConverter: backend.ASTConverter
