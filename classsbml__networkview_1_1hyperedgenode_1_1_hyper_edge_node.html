Detailed Description

A very simple class that is used to enable hyper edges in NetworkX graphs.

A hyper edge is an edge that connects more than two nodes. E.g. to display A+B => C visually, one needs a hyper edge.

NetworkX does not support hyper edges. Thus, hyper edges are reprsented by introducing an actual node between the end points of the hyper edge.

Parameters:
reaction,:A wrapped SBML reaction reaction: SBMLEntity
label,:A label that can be drawn onto this special node label: str
