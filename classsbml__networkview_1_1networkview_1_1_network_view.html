Detailed Description

The NetworkView is sort of a wrapper around the Qt Canvas concept with QGraphicsView and QGraphicsScene.

It inherits from QAbstractItemView to live in the world of Qt Model View Controller (MVC). Basically, it is there to bring these two worlds together. (Why the hell Qt does not provide something like this, I don't know.)

NetworkView handles the usual model update methods (e.g. dataChanged) and handles selection changes between the model-based views and the internal GraphicsView. (Which itself is not a QGraphicsView directly but a simple wrapper to support mousewheel zooming, etc.)

def sbml_networkview::networkview::NetworkView::redraw (   self)

Rewrite of the previous redraw approach.

It is based on the node positions as stored in the SBMLEntitys' Position property. Nodes and Edges are only created once (on drawGraph). They are not recreated here, only repositioned.
