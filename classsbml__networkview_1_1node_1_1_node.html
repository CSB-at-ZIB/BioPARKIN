Detailed Description

Represents a node in the GraphicsView.

When drawing the (networkx) graph every wrapped Species is put into a Node to facilitate drawing.

Parameters:
position,:Position of the Node position: Tuple of x,y floats
scene,:The corresponding Scene scene: QGraphicsScene
style,:Drawing style/brush for the node style: QBrush
rect,:Defines the size of the Node rect: QRect
matrix,:A transformation matrix matrix: QMatrix
wrappedObject,:Object represented by this node wrappedObject: SBMLEntity
