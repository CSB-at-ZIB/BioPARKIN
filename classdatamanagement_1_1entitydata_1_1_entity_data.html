Detailed Description

This serves as a data container for experimental and simulation data.

The data is in two lists one for time points (more generally called "Data Descriptors", one for data points.

Some properties give more information about the encapsulated data (e.g. the referenced libSBML Species/Parameter). Some convenience methods are provided (like .getId() which works on the referenced SBML entity).
