Detailed Description

This is a simple description of an Assignment Rule.

It is wrapped to be used by a computation backend.

Parameters:
index,:A unique index for every AssignmentRule has to be provided. index: int
mathNode,:A libSBML ASTNode root node mathNode: ASTNode
rule,:A libSBML rule object rule: libSBML rule (AssignmentRule)
