Detailed Description

This is a simple description of an ODE.

If the ODE is based on a libSBML Rule, the Rule is wrapped. An ODE can also result out of the Reactions between Species, in which case there is no explicit Rule to wrap.

The ODE is wrapped to be used by a computation backend.

Parameters:
index,:A unique index for every ODE has to be provided. index: int
mathNode,:A libSBML ASTNode root node mathNode: ASTNode
formula,:A math string in FORTRAN syntax formula: str
rule,:A libSBML rule object rule: libSBML rule (AssignmentRule, RateRule, ...)

def odehandling::odewrapper::ODEWrapper::mathForFortran (   self)

Takes the current libSBML mathNode AST tree and converts it into something that is valid FORTRAN code.

(e.g. "pow(x,2)" becomes "x ** 2", etc.)

If "formula" was given during creation of this instance it is assumed that this formula string already is valid FORTRAN code. It will be returned unaltered.
