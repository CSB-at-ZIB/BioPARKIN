Detailed Description

This is just kind of a "renamed" OrderedDict object to make the naming of things more coherent, to adhere to the standard set by the SBML Multiple Parameter Sets annotation and to have a place for functionality for which a need might arise.

A Parameter's "combined Id" (using Reaction ID and Param ID) is used as a key for every Parameter object.
