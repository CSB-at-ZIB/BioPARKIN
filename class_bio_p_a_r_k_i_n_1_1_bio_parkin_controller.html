Detailed Description

This is the heart of BioParkin.

It sets up the general UI (MainWindow, some Views, ...), handles creation and management of individual network windows, and instantiates needed services (for easy access to data, status bar, ...).

This class inherits from QMainWindow for its functionality and from the (automatically generated) Ui_MainWindow for the design of the UI (including the self.setupUi() method).

Loads a given SBML file, thereby creating a new subwindow on the main MDI Area.

If filename is None, it creates a new model.

There are a lot of reference variables, so that the system knows the currently active subwindow, etc.
