  parent,

)

Called from parent() test.

A self.model that returns an index of parent X should also return X when asking for the parent of the index

This recursive function does pretty extensive testing on the whole self.model in an effort to catch edge cases.

This function assumes that rowCount(QtCore.QModelIndex()), columnCount(QtCore.QModelIndex()) and index() already work. If they have a bug it will point it out, but the above tests should have already found the basic bugs because it is easier to figure out the problem in those tests then this one
