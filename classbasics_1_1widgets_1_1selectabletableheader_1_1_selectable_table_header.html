Detailed Description

A custom table header view that adds a QCheckBox to the left side of the header.

It can be used to (logically) select columns.

Whenever some selection in the supplied "selection model" is changed, we have to check whether this change affects one of self._isSectionSelected.

That depends on whether self has vertical or horizontal orientation.

We also have to check the state within the given "selection model" to assess whether the internal sections are completely checked/unchecked or in an intermediate state. (To be able to do this, self._isSectionSelected has to store "more" than just a bool. We now use a the Qt.Checkstate enum which has Qt.PartiallyChecked.)

The indices given by the two QModelIndex correspond to the internal sections by design.
