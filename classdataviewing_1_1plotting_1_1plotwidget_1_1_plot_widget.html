Detailed Description

A widget for plotting experimentalData.

It's still in a rough state and will be improved over time.

Internally, it uses matplotlib to easily draw plots. This is handy but probably rather performance-demanding. One future option for speeding things up could be to switch the plotting library (possibly Qwt).
