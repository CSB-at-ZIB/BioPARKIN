string sbml_model::parameter_sets::__author__ = "Moritz Wade"

This class provides the data structures for storing multiple Parameter Sets in one SBML file.

It's modelled after a semi-official annotation for SBML: http://www.sys-bio.org/sbwWiki/_media/sysbio/sysbio/parametersets/2010-02-22_-_managingmultipleparametersets.pdf

Objects of these classes will be saved as nested XML within a SBML Model's Annotation element.
