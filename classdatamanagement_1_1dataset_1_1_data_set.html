Detailed Description

This class encapsulates all the data that is in an individual data file, either simulation or experimental data.

It contains a stable dictionary of EntityData objects. Each such object holds the data for one entity (usually a SBML Species).

def datamanagement::dataset::DataSet::getNumOfRealData (   self)

Walks through all EntityData objects this DataSet holds and counts the number where isMetaData = False *and* isWeightData = false.

Those are the EntityData objects with "usable" data (i.e. that belong to a Species).
