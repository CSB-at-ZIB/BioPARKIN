Detailed Description

This service is initialized with a status bar upon which it sets a progress bar.

Consumers of the service can then show/hide the progress bar and set values.

The __new__ method is overridden to make this class a singleton. Every time it is instantiated somewhere in code, the same instance will be returned. In this way, it can serve like a static class.

Parameters:
statusbar,:The status bar on which to display progress. statusbar: QStatusBar (I think)

Creates a QProgressBar and adds it to the given status bar (for threads that can provide accurate progress information).

Note: The status bar can either be a QStatusBar or the StatusBarService. In the default BioPARKIN use, it is the StatusBarService instance.
